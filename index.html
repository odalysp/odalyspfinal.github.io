What is a smart city?

Using technology, smart cities create more efficient operating cities that also aim to reduce citiesâ€™ carbon footprints. Through the technologies, smart cities analyze collected data to improve the life of citizens. Some examples of smart cities around the world include Singapore, Oslo, Copenhagen, Amsterdam, etc. Technologies that are used in smart cities include information and communications tech, connected physical devices using the internet of things, and geographical information systems.
